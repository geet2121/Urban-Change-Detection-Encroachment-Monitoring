# Urban Change Detection – Encroachment Monitoring

This project detects land use classes from aerial images using OpenCV.

Dataset used:
UC Merced Land Use Dataset

Classes detected:
- Buildings
- Roads
- Vegetation
- Water

Techniques used:
- Image segmentation
- HSV color detection
- OpenCV image processing

Libraries:
Python, OpenCV, NumPy, Matplotlib

Output:
The system calculates the percentage area occupied by each land class in an aerial image.

Example Output:

Vegetation: 72%
Buildings: 15%
Roads: 8%
Water: 5%
